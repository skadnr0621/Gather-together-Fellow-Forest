| 구분 | 기능                    | 설명                                                         |  비고   |
| :--: | ----------------------- | ------------------------------------------------------------ | :-----: |
|  1   | 자유 게시판             | 자유롭게 글을 쓸 수 있는 게시판입니다. <br />친구를 찾거나, 그때 하지 못했던 말들을 할 수 있습니다. |         |

|  4   | 채팅                    | 동시 접속한 유저들과 함께 채팅으로 의사소통을 할 수 있습니다. | 수정 중 |
|  5   | 사진 촬영               | 찰칵! 메타버스 내에서 만난 친구들과 학교를 배경으로 사진을 찍을 수 있습니다. |         |
